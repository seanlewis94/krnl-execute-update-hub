## About KRNL

KRNL is a Windows desktop program built to run custom Lua scripts inside Roblox. As a separate client-side tool, it connects to the Roblox application so users can execute third-party scripts for automation, testing, or gameplay changes. The design focuses on simplicity, with a one-click injection flow and an integrated script library.

Made for both newcomers and power users, KRNL combines a dependable execution flow with local persistence and automatic updates. It keeps storage use low while still offering a multi-language interface and batch processing support. Since it is free and does not need key activation, it provides an easy way to get started with Lua scripting in Roblox.
